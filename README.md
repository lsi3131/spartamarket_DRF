# 스파르타마켓
SPARTA 내일배움캠프에서 과제로 제작한 프로잭트입니다.
자유롭게 물건을 등록하고 조회할 수 있습니다.

## 주요 기능
- 로그인/로그아웃/회원가입
- 프로필 기능
  - 아이디, 가입일, 등록 물품 확인
  - 찜한 물건 목록 조회
  - 팔로우 기능
- 게시 기능
  - 물건 목록과 디테일 페이지
  - 조회, 등록, 수정, 삭제 기능
  - 물건별 좋아요 기능

<br>

## 추가 기능 
- 회원 기능
  - 프로필 사진 수정 가능
- 게시 기능
  - 좋아요, 조회수 보여주기
  - 조회수, 인기도, 최신 순으로 조회 가능
- 검색 기능
  - 제목, 설명, 유저네임에 따라 검색 가능
